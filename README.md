# Exploratory Data Analysis on McDonald's Menu Dataset

## Objective
To analyze nutritional values of McDonald's menu items and identify trends and insights.

## Steps Performed
- Loaded and cleaned dataset
- Descriptive statistics
- Category and calorie analysis
- Visualization using bar charts and heatmaps
- Generated insights and recommendations

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Google Colab

## Outcome
The analysis revealed high-calorie and low-calorie food items, helping customers make healthier food choices.
